## üîç Search Guide

### Finding a Specific Animation Class

**Step 1**: Open `ANIMATIONS_QUICK_REFERENCE.md`  
**Step 2**: Search for the class name (Ctrl+F)  
**Step 3**: View the example and usage

**Common Classes**:
- `.fade-in` - Content fades in
- `.slide-in-left` - Slides in from left
- `.slide-in-right` - Slides in from right
- `.card` - 3D hover effect (automatic)
- `.btn` - Ripple effect (automatic)
- `.glow` - Glowing text
- `.bounce` - Bouncing animation

### Finding a Specific JavaScript Function

**Step 1**: Open `ANIMATIONS_QUICK_REFERENCE.md`  
**Step 2**: Search for "JavaScript Functions" section  
**Step 3**: Find the function you need

**Common Functions**:
- `notify(message, type)` - Show notifications
- `animateCounter(element, target)` - Animate counters

**Step 2**: Find the page type you're working on

- Dashboard
